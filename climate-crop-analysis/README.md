# Climate Crop Analysis (Rainfall vs Yield)

**Objective:** Explore correlation between annual rainfall and crop yield using sample data.

**Files**
- `rainfall_crop.ipynb` — Jupyter notebook
- `requirements.txt`
- sample data: `../assets/data/crop_rainfall_sample.csv`

**How to run**
1. pip install -r requirements.txt
2. jupyter lab → open `rainfall_crop.ipynb`
