# EDA & Data Cleaning

**Objective:** Demonstrate reproducible cleaning steps and a small pipeline converting raw -> cleaned -> summary.

**Files**
- `cleaning_pipeline.ipynb` — notebook
- `cleaning_pipeline.py` — script to run the cleaning
- sample data: `../assets/data/gps_sample.csv` or any CSV

**How to run**
1. pip install -r requirements.txt
2. jupyter lab -> open `cleaning_pipeline.ipynb`
